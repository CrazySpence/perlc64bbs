# perlc64bbs
Template for a C64 BBS written in perl

This perl file as is will have a welcome menu for new/returning users
A login screen
a Menu for the BBS with some default tools like date/time, connected users, messages
A game menu with a high low dice game

From this template it should be relatively easy to expand, my intention is to rope my Doomsday game into it for Commodore 64 users but i want to leave the clean template seperate on its own before I start mucking around with it any further
